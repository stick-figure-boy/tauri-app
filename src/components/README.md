# Components

In this directory, place components that can be used globally.
Component files describe only the DOM.
The business logic is described in the `hooks.ts` file and the style in `style.ts`.

```
ex)

components
  └── from
        └── textFrom
              ├── TextFrom.tsx
              ├── hooks.test.ts
              ├── hooks.ts
              └── style.ts
```
